# Documentation RGPD - Royaume des Paraiges

Cette documentation recense les elements de conformite RGPD pour l'application Royaume des Paraiges.

## Contexte reglementaire

Le **Reglement General sur la Protection des Donnees (RGPD)** s'applique a toute application collectant des donnees personnelles de residents europeens. La **CNIL** a publie des recommandations specifiques pour les applications mobiles (mise a jour avril 2025) et mene une campagne de controles depuis le printemps 2025.

| Droit de suppression | A faire | Procedure a definir |
| Droit de portabilite | A faire | Procedure a definir |
| Securite des donnees | Partiel | RLS active sur Supabase |

## Actions prioritaires

1. **Rediger la politique de confidentialite** et l'integrer dans l'application
2. **Implementer le consentement explicite** avant toute collecte de donnees
3. **Definir les durees de conservation** pour chaque type de donnee
4. **Mettre en place les procedures** d'exercice des droits utilisateurs
5. **Documenter les mesures de securite** existantes
